# ForgeUrFit Website

Complete static website for ForgeUrFit.

## What's included
- Home page
- Products page with filters
- Custom order builder
- Mission/about page
- Pricing and cost breakdown page
- FAQ page
- Contact/order inquiry page
- Cart drawer and local bag storage
- Mobile-responsive layout

## How to view locally
Open `index.html` in your browser.

## How to deploy on Vercel
Option 1: Upload the folder to GitHub, then import the repository in Vercel.

Option 2: Use Vercel CLI from this folder:

```bash
npm i -g vercel
vercel login
vercel
vercel deploy --prod
```

## Important edits before final launch
- Replace `orders@forgeurfit.com` in `script.js` with the real business email.
- Replace or update product prices if the final teacher-approved prices change.
- Add real product photos when available.
